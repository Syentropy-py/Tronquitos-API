## 💡 TIPS IMPORTANTES

1. **Lee en orden**
   - No saltes a producción sin testing
   - El orden importa para entender

2. **Testa mucho**
   - Cada paso tiene un test
   - No avances al siguiente sin que un paso funcione

3. **Guarda todo**
   - Copia los IDs de webhook
   - Guarda URLs en un documento seguro
   - Backups de prompts que funcionen

4. **Documento vivo**
   - Estos archivos se actualizan
   - Si ves un error, reporta
   - Mejora los prompts basado en feedback

## 📌 NOTAS IMPORTANTES

### Sobre Testing vs Producción
```
TESTING (donde estás ahora):
- Localhost: http://localhost:5678
- Base de datos local
- Sin SSL requerido
- Puedes debuggear fácil

PRODUCCIÓN (luego):
- Cloud N8N o servidor
- HTTPS obligatorio
- Credenciales en secretos
- Monitoreo crítico
```

### Sobre Prompts
```
NO necesitas el "mejor prompt" de día 1.
Comienza con PROMPT 1 (simple).
Testa y mide resultado.
Mejora semana a semana basado en feedback.
```
